# UpStartUp-official
Platform for lean startup development.

## Installation

* `$ git clone git@github.com:bogomazov/UpStartUp-official.git`
* `$ virtualenv upstartup-env`
* `$ cd upstartup/`
* `$ pip install -r requirements.txt`
* `$ bower install`
* `$ python manage.py migrate`
* `$ python manage.py runserver`
